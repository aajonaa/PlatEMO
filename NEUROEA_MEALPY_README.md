# NeuroEA: Neural Network-guided Evolutionary Algorithm

Pure Python implementation of NeuroEA with Mealpy native integration.

**Transfer-learned from CEC2017 Stage 2 training (F1 → F9, Dimension=30)**

## Overview

NeuroEA is an evolutionary algorithm guided by neural network principles, implemented as a native **Mealpy optimizer** class. It features an 11-block modular architecture that can be easily customized and extended.

### Key Features

- ✅ **Native Mealpy Integration** - Seamless integration with Mealpy's optimization framework
- ✅ **Transfer Learning** - Pre-trained parameters from CEC2017 benchmark suite
- ✅ **11-Block Architecture** - Modular design with tournament selection, information exchange, crossover, and mutation
- ✅ **Full Validation** - Input validation using Mealpy's validator system
- ✅ **No External Dependencies** - Requires only NumPy and Mealpy
- ✅ **Production Ready** - Tested on CEC2017 constrained optimization benchmarks

## Quick Start

### Basic Usage
```python
import numpy as np
from mealpy import FloatVar
from NeuroEA import OriginalNeuroEA

def objective_function(solution):
    return np.sum(solution**2)

problem_dict = {
    "bounds": FloatVar(n_vars=30, lb=(-10.,)*30, ub=(10.,)*30, name="x"),
    "obj_func": objective_function,
    "minmax": "min",
}

model = OriginalNeuroEA(epoch=100, pop_size=30, c1=0.5, m1=0.1)
g_best = model.solve(problem_dict)
print(f"Best fitness: {g_best.target.fitness}")
```

### Using Transfer-Learned Parameters
```python
from NeuroEA import TrainedNeuroEA

# Automatically loads trained parameters from CEC2017 training
model = TrainedNeuroEA(epoch=100, pop_size=30)
model.information()  # Display training details
g_best = model.solve(problem_dict)
```

## Algorithm Architecture (11 Blocks)

```
┌─────────────────────────────────────────────────────────┐
│ Block P (Population): Maintain current population       │
└──────────────────┬──────────────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────────────┐
│ Blocks T1-T3 (Tournament): Select individuals 3x        │
│   - Random tournament from population                   │
│   - Keep best from tournament                           │
│   - Repeat 3 times for diversity                        │
└──────────────────┬──────────────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────────────┐
│ Blocks E1-E4 (Exchange): Information exchange           │
│   - Divide selected into 4 groups                       │
│   - Share promising solutions between groups            │
└──────────────────┬──────────────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────────────┐
│ Block C (Crossover): Recombination                      │
│   - Blend two selected individuals                      │
│   - Arithmetic crossover with rate c1                   │
└──────────────────┬──────────────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────────────┐
│ Block M (Mutation): Variation                           │
│   - Gaussian mutation with rate m1                      │
│   - Adaptive sigma based on search space bounds         │
└──────────────────┬──────────────────────────────────────┘
                   │
┌──────────────────▼──────────────────────────────────────┐
│ Block S (Selection): Survival selection                 │
│   - Merge parents and offspring                         │
│   - Keep best pop_size individuals                      │
│   - Sort by fitness                                     │
└──────────────────┬──────────────────────────────────────┘
                   │
                   └──────────────────────────────┐
                                                  │
                                    Next generation
                                         (P)
```

## Hyperparameters

| Parameter | Type | Default | Range | Description |
|-----------|------|---------|-------|-------------|
| `epoch` | int | 100 | [1, 100000] | Number of iterations |
| `pop_size` | int | 30 | [5, 10000] | Population size |
| `c1` | float | 0.5 | [0, 1] | Crossover rate (Block C) |
| `m1` | float | 0.1 | [0, 1] | Mutation rate (Block M) |
| `tournament_size` | int | 10 | [2, 100] | Tournament size (Blocks T1-T3) |

## Classes

### OriginalNeuroEA
Standard NeuroEA implementation with customizable hyperparameters.

```python
model = OriginalNeuroEA(
    epoch=100,          # Number of generations
    pop_size=30,        # Population size
    c1=0.5,            # Crossover rate
    m1=0.1,            # Mutation rate
    tournament_size=10  # Tournament size
)
```

### TrainedNeuroEA
Pre-configured with parameters learned via transfer learning on CEC2017 benchmark.

```python
model = TrainedNeuroEA(
    epoch=100,
    pop_size=30,
    c1=None,  # Load from trained_neuroea_params.json
    m1=None,  # Load from trained_neuroea_params.json
    tournament_size=10,
    params_file='trained_neuroea_params.json'  # Path to trained parameters
)

# Display training information
model.information()
```

## Methods

### generate_agent(solution)
Create an agent with fitness evaluation.
```python
agent = model.generate_agent(solution)
```

### tournament_selection(pop_indices, tournament_size)
Select best individual from random tournament.
```python
winner_idx = model.tournament_selection([0, 1, 2, 3], tournament_size=2)
```

### crossover_operator(parent1, parent2, crossover_rate)
Blend two solutions via arithmetic crossover.
```python
child = model.crossover_operator(parent1, parent2, c1=0.5)
```

### mutation_operator(solution, mutation_rate)
Apply Gaussian mutation to solution.
```python
mutated = model.mutation_operator(solution, m1=0.1)
```

### amend_solution(solution)
Clip solution to problem bounds.
```python
valid_solution = model.amend_solution(out_of_bounds_solution)
```

## Examples

### Example 1: Basic Optimization
```python
from NeuroEA import OriginalNeuroEA
from mealpy import FloatVar
import numpy as np

# Objective function
def sphere(x):
    return np.sum(x**2)

# Problem definition
problem = {
    "bounds": FloatVar(n_vars=30, lb=(-10.,)*30, ub=(10.,)*30),
    "obj_func": sphere,
    "minmax": "min",
}

# Optimize
model = OriginalNeuroEA(epoch=100, pop_size=30)
best = model.solve(problem)
print(f"Best: {best.target.fitness:.6e}")
```

### Example 2: Using Trained Parameters
```python
from NeuroEA import TrainedNeuroEA

# Automatically loads CEC2017 trained parameters
model = TrainedNeuroEA(epoch=100, pop_size=30)

# Show what was learned
model.information()

# Solve problem with transfer-learned parameters
best = model.solve(problem)
```

### Example 3: Hyperparameter Tuning
```python
from NeuroEA import OriginalNeuroEA

# Try different crossover rates
for c1 in [0.3, 0.5, 0.7, 0.9]:
    model = OriginalNeuroEA(epoch=50, pop_size=30, c1=c1, m1=0.1)
    best = model.solve(problem)
    print(f"c1={c1}: fitness={best.target.fitness:.6e}")
```

## Comparison with Other Algorithms

NeuroEA can be compared with other Mealpy optimizers:

```python
from mealpy import PSO, GA, DE
algorithms = [
    OriginalNeuroEA(epoch=100, pop_size=30),
    PSO.OriginalPSO(epoch=100, pop_size=30),
    GA.BaseGA(epoch=100, pop_size=30),
    DE.OriginalDE(epoch=100, pop_size=30),
]

for algo in algorithms:
    best = algo.solve(problem)
    print(f"{algo.__class__.__name__}: {best.target.fitness:.6e}")
```
